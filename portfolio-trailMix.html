TrailMix Project

Description

  • Live Site

  • GitHub

TrailMix is a hiking trail-finding based web application built by a team of 5 developers during a 48 hour hackathon. The project was originally completed utilizing JavaScript and jQuery, and was later ported over to ReactJS and Redux.

Technology

  • ReactJS
  • Redux
  • Google Maps API
  • Google Directions API
  • Google Autocomplete API
  • REI API
