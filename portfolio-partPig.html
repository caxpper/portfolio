Part Pig Project

Description

  • Live Site

PartPig is an ecommerce platform for selling used car parts. PartPig exists to fill the gap in connecting the buyer and the seller in a way that is user friendly and convenient.

It was completed by a team of 5 developers. We used Agile methodologies to create an increasingly more complex product and Git as version control. Jira was the central hub for create the sprints, delegating tasks, reporting bugs, create diagrams, etc.

The application is still in a Beta version but in the future would be released for a real use.
